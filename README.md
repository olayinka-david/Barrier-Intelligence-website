# Barrier Intelligence Ceramic template

A responsive multi-page marketing website for Barrier Intelligence and Assure3D. The template uses the published Ceramic Design System package, the `barrier-intelligence` theme, semantic color roles, logical CSS properties, light and dark color modes, reduced-motion handling, keyboard-operable controls, and RTL-aware layout and motion.

## Run locally

```sh
npm install
npm run dev
```

Open `http://localhost:4173/`. Add `?theme=dark` to request dark mode and `?dir=rtl` to exercise the RTL layout.

## Build and preview

```sh
npm run build
npm run preview
```

## Ceramic validation

```sh
npx utopia-ds template validate .
```
